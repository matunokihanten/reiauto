🍽 松乃木飯店 Pro Ultimate
リアルタイム座席管理システム

このプロジェクトは、飲食店の座席状況（空席、使用中、予約など）をリアルタイムで管理・共有するためのWebアプリケーションです。スタッフは直感的なドラッグ＆ドロップでレイアウトを編集でき、お客様はスマートフォンから現在の空席状況を「閲覧専用」で確認できます。

🌟 主な機能
直感的なレイアウト編集: 編集モードで席や壁、アイコンを自由に配置。

リアルタイム同期: Socket.io を使用し、複数の端末で操作内容を瞬時に同期。

3つの状態管理: 「空席」「使用中」「まもなく空き」「予約」を色別で表示。

自動タイマー: 着席からの経過時間を計測し、設定時間を過ぎると「まもなく空き」へ自動変更。

完全な公開モード: お客様用URLでは編集機能を完全にロックし、閲覧のみを許可。

多フロア対応: 1階、2階など複数のフロアを切り替えて管理可能。

🛠 使用している技術
Frontend: HTML5, CSS3, JavaScript (Vanilla JS)

Backend: Node.js

Real-time: Socket.io

Hosting: Render (推奨リージョン: Singapore)

🚀 導入手順（デプロイ方法）
初級者の方でも、以下の手順で自分のサーバーを立ち上げることができます。

1. 準備
GitHubのアカウントを作成し、提供された index.html および server.js などのファイルをリポジトリ（保存庫）にアップロードしてください。

2. Renderでの設定
Render にログインし、「+ New」→「Web Service」 を選択します。

作成したGitHubのリポジトリを接続します。

Region（リージョン）の設定:

ここが動作の軽快さに直結します。必ず 「Singapore (Southeast Asia)」 を選択してください。

Runtime: Node を選択します。

Build Command: yarn install または npm install

Start Command: node server.js

3. デプロイ完了
「Your service is live」と表示されたら、発行されたURLにアクセスして動作を確認してください。

📖 使い方
👤 スタッフ用（管理画面）
URL: [https://あなたのサービス名.onrender.com/](https://あなたのサービス名.onrender.com/)

操作:

「🔨 編集」ボタンでレイアウト変更。

席をクリックしてステータス変更。

右クリック（または長押し）で詳細情報を入力。

👥 お客様用（公開画面）
URL: [https://あなたのサービス名.onrender.com/?public=true](https://あなたのサービス名.onrender.com/?public=true)

特徴:

閲覧専用 です。編集ボタンや管理メニューは一切表示されません。

ドラッグによる移動や範囲選択もプログラム側でブロックされています。

📂 ファイル構成
index.html: アプリケーションの本体（見た目と動きのすべて）。

server.js: 通信を仲介するバックエンドプログラム。

package.json: 必要なライブラリのリスト。

⚠️ 運用上の注意
配置保存: レイアウトを変更した後は、必ず「💾 保存（ホーム保存）」を押してください。これにより、次回起動時や「配置リセット」時にその位置が基準となります。

データのバックアップ: 「📥 データ書き出し」ボタンを使って、定期的に設定ファイルをパソコンに保存しておくことをお勧めします。

🔧 カスタマイズ
index.html 内の以下の変数を書き換えることで、設定を変更できます。

TIMER_MS: 同期やタイマーの更新間隔。

SEAT_COLORS: 席の色を変更可能。

ICONS: 追加できる絵文字アイコンのリスト。
